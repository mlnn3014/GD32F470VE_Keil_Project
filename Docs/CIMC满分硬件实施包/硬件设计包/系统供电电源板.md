# 系统供电电源板

## 目标

- 板名：系统供电电源。
- 芯片：GD30DC1354。
- 输入：18V~36V DC，典型 24V。
- 输出：5.00V/2A，给 GD32F470 开发板 5V 供电点。
- 输出精度验收：空载和开发板负载均为 5V ±0.15V。

## 关键计算

反馈电压为 0.6V：

```text
Vout = 0.6V * (1 + Rtop / Rbottom)
     = 0.6V * (1 + 110k / 15.0k)
     = 5.00V
```

15uH 电感纹波电流：

```text
Delta_IL = (Vin - Vout) * Vout / (Vin * Fsw * L)
Fsw = 500kHz, L = 15uH, Vout = 5V

Vin = 18V: Delta_IL = 0.48A, Ipeak = 2.24A
Vin = 24V: Delta_IL = 0.53A, Ipeak = 2.26A
Vin = 36V: Delta_IL = 0.57A, Ipeak = 2.29A
```

输入电容耐压：

```text
Vin_max = 36V
电解使用 63V，陶瓷使用 100V，满足降额要求。
```

输入电容纹波电流最坏约为输出电流一半，电容纹波能力按 `>=1A RMS` 选型更稳。

## PCB 满分检查

- U1、C2/C3/C4、L1、C7/C8 必须围绕芯片紧凑放置。
- SW 铜皮只覆盖 U1 SW 到 L1 的必要面积，不穿过反馈区。
- R3/R4 紧贴 FB 脚，FB 线从输出电容正端 Kelvin 取样。
- 输入大电流回路和输出大电流回路走线短、宽，顶层和底层并联铺铜，过孔阵列连接。
- 底层尽量完整 GND，不在芯片地、输入电容地、输出电容地之间切割。
- `I-MEAS` 串联接口位置靠近输出端子，正常用短接帽。
- 顶层覆铜文字：`系统供电电源 队伍编号 2026-05`。

## 调试步骤

1. 不焊 U1 前确认 VIN 到 GND 无短路。
2. 焊接完成后先用限流电源 18V/100mA 上电，确认无异常发热。
3. 空载测 18V、24V、36V 输入时 5V 输出。
4. 接开发板负载测 18V、24V、36V 输入时 5V 输出。
5. 记录输出电压、电流、U1 温升、电感温升，填入 `测试记录模板.md`。
