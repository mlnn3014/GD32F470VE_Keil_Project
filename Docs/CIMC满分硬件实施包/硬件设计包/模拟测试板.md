# 模拟测试板

## 目标

- 板名：模拟测试板。
- 功能：用板载精密电阻替代三线制 PT100，模拟不同温度点。
- 挡位：12 挡，覆盖基础 6 点和进阶 6 点。
- 精度：所有电阻 `<=0.1%`，建议 `25ppm/°C` 或更低。

## 原理图连接

| 模块 | 位号建议 | 参数 | 连接/说明 |
| --- | --- | --- | --- |
| 输出端子 | J1 | 3Pin | `WHITE`, `RED_FORCE`, `RED_SENSE`，顺序与采样板一致 |
| 旋转开关 | SW1 | 1P12T, break-before-make | 公共端接 `WHITE`，12 个选择端分别接一个精密电阻 |
| 精密电阻 | R1~R12 | 0.1%, 25ppm/°C | 每个电阻另一端接 `RED_FORCE/RED_SENSE` Kelvin 公共点 |
| 测试点 | TP1~TP3 | 焊盘 | `WHITE`, `RED_FORCE`, `RED_SENSE` |

红线 Kelvin 接法：

```text
RED_FORCE 和 RED_SENSE 在被选电阻的同一端汇接。
WHITE 经旋转开关选择某一个电阻的另一端。
任何挡位只有一个电阻位于 WHITE 与 RED 公共点之间。
```

必须使用先断后合旋转开关，避免切换瞬间两个电阻并联。若采购的旋转开关无法保证先断后合，改用 12 组独立跳线帽，但结构上同一时间只能插一个跳线帽，并在丝印上写 `ONLY ONE JUMPER`。

丝印格式建议：

```text
1 -49.26C 80.60R
2 -44.47C 82.50R
...
12 141.11C 154.00R
```

若实际采购阻值与标称不同：

1. 用四位半或六位半万用表测实际阻值。
2. 按实际阻值换算理论温度。
3. 修改丝印或在板上留白贴标签。
4. 更新方案书、测试记录和固件 `pt100_lut[]`。

## PCB 满分检查

- 旋转开关居中，12 个电阻围绕开关短线连接。
- 电阻到端子的走线宽而短，建议 20mil 或更宽。
- WHITE 走线只经过开关公共端，不绕行。
- RED_FORCE 与 RED_SENSE 到电阻公共端尽量等长、并行，形成 Kelvin 汇接。
- 每个挡位旁丝印清楚可读，温度和阻值都不能省。
- 输出端子丝印与采样板完全一致：`WHITE`, `RED_FORCE`, `RED_SENSE`。
- 顶层覆铜文字：`模拟测试板 队伍编号 2026-05`。

## 出厂前检查

| 检查项 | 要求 |
| --- | --- |
| 开关挡位 | 每挡只有一个电阻导通 |
| 相邻挡位 | 不存在两个电阻并联 |
| 端子顺序 | 与采样板完全一致 |
| 实测阻值 | 每挡记录到测试表 |
| 丝印 | 每挡温度和阻值清晰 |
