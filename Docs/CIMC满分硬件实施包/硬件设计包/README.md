# CIMC 初赛满分硬件实施包

本目录把满分冲刺硬件方案拆成可以直接用于嘉立创 EDA 建板、方案书编写、测试视频录制的工程资料。

## 交付文件

- `系统供电电源板.md`：GD30DC1354 18V~36V 转 5V/2A 电源板原理图、BOM、计算和 PCB 检查表。
- `PT100温度采样板.md`：GD30AD3344 PT100 采样主板原理图、模拟前端、SPI 接口、标定和 PCB 检查表。
- `模拟测试板.md`：12 挡精密电阻模拟 PT100 测试板设计、阻值表、切换机构和标定要求。
- `测试记录模板.md`：电源、SPI、基础精度、进阶精度和视频录制检查表。

## 建板原则

1. 三块板独立建工程、独立打样、独立导出 Gerber/坐标/BOM。
2. 嘉立创 EDA 专业版 V3.2 建板，双层 PCB。
3. 每块板顶层覆铜写板卡名称、队伍编号、设计年月。
4. 丝印不出现学校、学院、队员姓名、logo、缩写或任何身份信息。
5. 先按本文档完成原理图 ERC、PCB DRC，再下单打样。

## 固件配套

本工程固件已按该硬件方案配置 GD30AD3344：

- SPI3：`PE2=SCK`、`PE5=MISO/DOUT_DRDY`、`PE6=MOSI/DIN`、`PE4=CS`。
- AIN0：PT100 调理电压，PGA `±2.048V`。
- AIN3：接 2.500V 精密基准，固件按官方 ExtRef 例程使能 AIN3 作为 ADC 外部参考源。
- 输出：OLED 显示 PT100 温度、ADC 原始值、ADC 电压、EXTREF 状态；USART0 每秒输出一行 PT100 数据。
