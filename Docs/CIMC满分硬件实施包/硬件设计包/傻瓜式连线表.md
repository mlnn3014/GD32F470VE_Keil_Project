# 傻瓜式 EDA 连线表

这份表按“在 EDA 里怎么画”来写。你可以按下面流程做：

1. 先新建 3 个工程：`系统供电电源`、`PT100 温度采样板`、`模拟测试板`。
2. 每个工程先放元件。
3. 给元件脚放网络标签 NetLabel。
4. 同名网络就是已经连在一起，不一定要用长线全部画出来。
5. 原理图画完后用“高亮网络”逐个检查：同名必须全亮，不同名不能误亮。

重要提醒：

- 元件库脚号可能不同，优先看脚名，例如 `VIN`、`GND`、`SW`、`FB`、`BST`、`SCLK`、`DIN`、`DOUT`、`AIN0`。
- 电源板的 `SW` 是噪声最大节点，只接芯片 SW、电感、自举电容，不要乱拉线。
- 采样板的 `AGND` 和 `DGND` 只通过一个 Net Tie 或 0R 单点连接。
- 测试板任意时刻只能接入一个电阻，不允许多个电阻并联。

### 1.4 PCB 摆放顺序

1. J1、F1、Q1、TVS1 放输入口附近。
2. U1 放中间，C2/C3/C4 紧贴 U1 的 VIN 和 GND。
3. L1 紧贴 U1 的 SW。
4. COUT1/COUT2/COUT3 紧贴 L1 输出端。
5. RFB_TOP/RFB_BOT/CFF 紧贴 U1 的 FB。
6. J2 和 J3 放输出侧，J2 串在 `VOUT_REG` 和 `VOUT_5V` 中间。

### 2.5 恒流源连线

这一块不同运放/三极管画法会有差异，EDA 里按下面网络目标画：

| 功能脚/节点 | 接到哪里 |
| --- | --- |
| U3A 电源正 | `3V3_A` |
| U3A 电源负 | `AGND` |
| U3B 电源正 | `3V3_A` |
| U3B 电源负 | `AGND` |
| RSET1 一端 | `VREF_2V5` 或恒流源参考节点 |
| RSET1 另一端 | U3A 反馈节点 |
| Q1 恒流输出 | `RED_FORCE` |
| U3A 控制输出 | Q1 控制脚 |
| RSET2 一端 | `VREF_2V5` 或恒流源参考节点 |
| RSET2 另一端 | U3B 反馈节点 |
| Q2 恒流输出 | `RED_SENSE` |
| U3B 控制输出 | Q2 控制脚 |

检查目标：

- `RED_FORCE` 上有约 1.000mA 激励电流。
- `RED_SENSE` 上有约 1.000mA 补偿电流。
- 两个恒流源都由 `VREF_2V5` 和 `2.500k` 精密电阻决定。

## 4. 最后检查清单

### 电源板

- `VIN_IN` 只在 J1 和 F1 前端出现。
- `VIN_PROT` 接到 U1 VIN、输入电容、TVS。
- `SW` 只接 U1 SW、L1、CBOOT。
- `FB` 只接 U1 FB、RFB_TOP、RFB_BOT、CFF。
- `EN` 不接 24V，只悬空或接测试点。
- J2 必须串在 `VOUT_REG` 和 `VOUT_5V` 之间。

### PT100 采样板

- `AGND` 和 `DGND` 只有 NT1 一个连接点。
- J2 三线顺序是 `WHITE`, `RED_FORCE`, `RED_SENSE`。
- GD30AD3344 AIN0 接 `PT100_ADC`。
- GD30AD3344 AIN3 接 `AIN3_REF`，由 `VREF_2V5` 通过 100R 过来，只作为外部参考源，不再接传感器信号。
- SPI 四根线都串 51R。
- J3 六针顺序是 `3V3_D`, `DGND`, `SPI_SCK`, `SPI_MISO_DRDY`, `SPI_MOSI_DIN`, `SPI_CS`。

### 模拟测试板

- SW1-COM 接 `WHITE_OUT`。
- 每个开关挡位只接一个电阻。
- 所有电阻另一端都接 `RED_KELVIN`。
- `RED_FORCE_OUT` 和 `RED_SENSE_OUT` 在 `RED_KELVIN` 同一点汇接。
- J1 三线顺序和采样板 J2 完全一致。
