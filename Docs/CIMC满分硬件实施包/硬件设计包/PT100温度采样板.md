# PT100 温度采样板

## 目标

- 板名：PT100 温度采样板。
- 核心 ADC：GD30AD3344，SPI 接外部 GD32F470 MCU。
- 输入：三线制 PT100 或模拟测试板。
- 采样范围：-50°C~+150°C。
- 基础精度：0°C~100°C 基础 6 点误差 <= ±10°C。
- 进阶精度：全量程 6 个进阶点经软件标定后误差 <= ±8°C。

## 系统接口

| 接口 | 位号建议 | 引脚 | 说明 |
| --- | --- | --- | --- |
| 5V 输入 | J1 | `5V`, `GND` | 从开发板或系统供电电源板取电 |
| PT100 输入 | J2 | `WHITE`, `RED_FORCE`, `RED_SENSE` | 三线制 PT100/模拟测试板同序连接 |
| SPI 排针 | J3 | `3V3`, `GND`, `SCK`, `MISO_DRDY`, `MOSI_DIN`, `CS` | 接 GD32F470 SPI3 |
| 测试点 | TP1~TP8 | `5V`, `3V3_A`, `3V3_D`, `2V5_REF`, `IEXC`, `ICOMP`, `AIN0`, `GND` | 调试和视频展示 |

推荐 MCU 接线：

| GD30AD3344 | 采样板 J3 | GD32F470 |
| --- | --- | --- |
| SCLK | SCK | PE2 / SPI3_SCK |
| DOUT/DRDY | MISO_DRDY | PE5 / SPI3_MISO |
| DIN | MOSI_DIN | PE6 / SPI3_MOSI |
| CS | CS | PE4 / GPIO |
| VDD | 3V3 | 3.3V |
| GND | GND | GND |

GD30AD3344 使用 SPI Mode 1：`CPOL=0`, `CPHA=1`。每条 SPI 线串联 `51Ω`。

## 原理图模块

### 电源与地

| 位号建议 | 参数 | 连接/说明 |
| --- | --- | --- |
| U1 | 低噪声 LDO, 5V->3.3V, Iout >= 150mA | 输入 5V，输出 `3V3_A` |
| FB1 | 磁珠或 0Ω | `3V3_A` 到 `3V3_D` |
| C1,C2 | 10uF/10V X7R | LDO 输入/输出 |
| C3,C4 | 0.1uF | ADC 和运放局部去耦 |
| AGND/DGND | 单点连接 | 在 GD30AD3344 附近用 0Ω 或 Net Tie 单点连接 |

### 2.500V 精密基准

| 位号建议 | 参数 | 连接/说明 |
| --- | --- | --- |
| U2 | 2.500V, 0.1%, 10ppm/°C 精密基准 | 输出 `VREF_2V5` |
| C5 | 1uF X7R | VREF 输出到 AGND |
| C6 | 0.1uF | 靠近 U2 |
| R1 | 100Ω | VREF 到 AIN3 外部基准输入限流 |

说明：本方案按大赛 `GD30AD3344_EVB_DemoSuites_ExtRef` 例程走满分稳妥路线：外部 2.500V 精密基准接到 AIN3，软件初始化时使能 `AIN3 as extern ref`，让 GD30AD3344 用 AIN3 作为 ADC 转换参考源。这个 2.500V 同时给恒流源和 0.4096V 偏置使用，形成同源基准。AIN3 一旦作为外部基准使用，就不要再当普通模拟输入通道采样。

### 三线制恒流激励与补偿

采用双匹配恒流源：

- `IEXC = 1.000mA`：从 `RED_FORCE` 流过 PT100 到 `WHITE`。
- `ICOMP = 1.000mA`：流过 `RED_SENSE` 补偿支路，使两根红线等阻时引线压降一阶抵消。

| 位号建议 | 参数 | 连接/说明 |
| --- | --- | --- |
| U3A/U3B | 零漂双运放，Vos <= 10uV，漂移 <= 0.1uV/°C | 两路恒流源控制 |
| Q1/Q2 | 小信号 P-MOS 或 PNP | 高边恒流输出 |
| RSET1/RSET2 | 2.500k, 0.1%, 25ppm/°C | `I = 2.500V / 2.500k = 1.000mA` |
| RIN1/RIN2 | 100Ω | 传感器输入保护限流 |
| CDIFF | 100nF C0G/X7R | WHITE 与 RED_SENSE 差模滤波 |
| CCM1/CCM2 | 10nF | WHITE/RED_SENSE 到 AGND 的共模滤波，靠近接口 |

设计文档写法：

```text
三线制 PT100 的两根红线等长，导线电阻近似相等。
双匹配恒流源让激励线和补偿线产生相等引线压降。
差分放大器取 WHITE 与 RED_SENSE 的补偿后电压，红线引线电阻一阶抵消。
模拟测试板为短线接入，剩余小误差由 12 点标定表吸收。
```

PT100 自热：

```text
Pmax = Rmax * I^2 = 157.33Ω * (1mA)^2 = 0.157mW
```

### 放大与偏置

前端目标：

```text
Vadc = 0.4096V + 16 * (I * Rpt100 - 80.31mV)
I = 1.000mA

-50°C: Rpt100 = 80.31Ω,  Vadc = 0.4096V
150°C: Rpt100 = 157.33Ω, Vadc = 1.64192V
```

该范围落在 GD30AD3344 `±2.048V` 单端正量程约 20%~80%，满足赛题要求。

推荐实现：

| 位号建议 | 参数 | 连接/说明 |
| --- | --- | --- |
| U4 | 零漂仪表放大器或三运放仪表放大电路 | 输入 WHITE/RED_SENSE 补偿后差分 |
| RG | 按所选芯片计算 | 增益设为 16 |
| U5A | 零漂运放 | 产生 0.4096V 偏置 |
| R2/R3/R4 | 0.1% 分压/缓冲 | 由 2.500V 基准分压得到 0.4096V |
| R5 | 1k | ADC 输入串联电阻 |
| C7 | 3.3uF | AIN0 到 AGND，`fc = 1/(2*pi*1k*3.3uF) ≈ 48Hz` |

如果使用具体仪表放大器，请在嘉立创 EDA 里按芯片 datasheet 的增益公式计算 `RG`，并把公式写进方案书。

### GD30AD3344 ADC

| GD30AD3344 引脚 | 连接 |
| --- | --- |
| VDD | `3V3_D`，旁路 `0.1uF + 1uF` |
| GND | DGND/单点地 |
| AIN0 | PT100 调理输出 |
| AIN1/AIN2 | 预留测试点或接中间电位，未用可悬空 |
| AIN3 | 2.500V 外部基准输入，经 100Ω 串联 |
| SCLK/CS/DIN/DOUT | SPI 排针，串 51Ω |

固件配置已匹配：

| 项目 | 配置 |
| --- | --- |
| 外部基准 | 初始化时按官方 ExtRef 例程写扩展寄存器，使能 AIN3 外部参考 |
| AIN0 | PT100 调理输出，PGA `±2.048V`，12.5SPS |
| AIN3 | 只接 2.500V 外部基准，不再作为普通 ADC 通道读取 |

## PCB 满分检查

- PT100 接口、恒流源、仪表放大器、ADC 按信号流直线放置。
- 模拟区域与 SPI 排针分区，SPI 不穿过模拟输入和反馈区域。
- WHITE/RED_SENSE 差分线短、并行、等长，靠近接口先滤波。
- 2.500V 基准远离电源板、电感和 SPI，周围用 AGND 护地。
- AIN0 RC 滤波贴近 ADC。
- ADC 去耦电容同层贴近 VDD-GND，不绕远。
- AGND/DGND 只在 ADC 附近单点连接。
- 顶层覆铜文字：`PT100 温度采样板 队伍编号 2026-05`。

## 调试步骤

1. 不插传感器，上电确认 3.3V 和 2.500V。
2. 测 `IEXC` 与 `ICOMP`，目标 1.000mA，偏差记录入方案书。
3. 接模拟测试板 100Ω 档，确认 AIN0 约 0.725V。
4. 用串口确认每秒输出 `PT100: raw=... adc=... temp=... extref=2500000uV ON`。
5. OLED 确认显示温度、RAW/ADC、EXTREF。
6. 逐个挡位测试并填写 `测试记录模板.md`。
